# User Manual

This document explains how to use the Evidence-Based Discourse Platform to 
participate in structured online debates. Setup information can be accessed in the README.md. The platform can be accessed locally or via the deployed version: https://discourse-gold.vercel.app/

## Purpose

The platform is designed to encourage structured, evidence-based discussion by:
- requiring users to state clear positions
- supporting arguments with evidence
- enabling evaluation based on factuality and reasoning
- facilitating structured debate

## Getting Started

1. Register a new account
2. Log in to the system
3. Navigate to the Explore page to view debates

## Creating a Debate

- Navigate to "Create Debate"
- Enter a question or topic
- Submit the debate

## Creating an Argument

- Select a debate
- Choose your stance (affirmative,opposing, neutral)
- Write your argument
- Attach at least one piece of evidence
- Submit

## Adding Evidence

Each argument must include at one piece of evidence, which includes:
- A mandatory URL
- Optional metadata (e.g. title, source type)

This ensures arguments are supported and verifiable.

## Replying to Arguments

- Select an existing argument
- Click "Reply"
- Add your response

## Voting on Arguments

Users can evaluate arguments based on:
- Factuality
- Soundness

Each dimension is rated independently.
Users cannot vote on their own arguments.

## Discussion Structure

Arguments are displayed in a threaded format:
- Replies are nested under parent arguments
- This makes discussions easier to follow
